User Crud

Esté es un CRUD (Create, Read, Update, Delete) muy sencillo que simula un centro de control de usuarios, permite la visualización de la información consumida desde una API pública realizando las peticiones mediante Axios. Además, es posible crear, leer la información, actualizar y eliminar cualquier usuario de la lista.

La aplicación fue desarrollada con React, haciendo uso de useForm, useState y useEffect.

▪ Aptitudes: Hojas de estilos en cascada (CSS) · HTML5 · JavaScript, React.js.

ENLACE:https://crud-user-nzerpa.netlify.app/

Poke App

Es una página que permite interactuar con todos los Pokémon existentes y que son consumidos desde la API oficial de Pokémon realizando las peticiones mediante Axios, también se utilizó React Router y su opción de rutas protegidas para forzar el registro virtual del usuario con su nombre. Está página contiene una opción de búsqueda por nombre del Pokémon con un campo de escritura y un filtro que permite elegir entre los 20 tipos diferentes de Pokémon, además es posible acceder a la información detallada de cada personaje.

Importante destacar el uso de CCS, React Redux, React Router, useSelect y los hooks habituales useEffect y useState

Enlace: https://poke-api-nzerpa.netlify.app/
